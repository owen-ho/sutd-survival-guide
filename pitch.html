How It's Made

Four cohort projects, unified behind one bot.

A single token + dispatcher routes every tap by a namespace:action scheme, so each feature stays its own clean module.

CorePython · python-telegram-bot (async, JobQueue for reminders)
DataSQLite (WAL) — shared modules, per-user reminder state, ACID writes
Live datahttpx — arrivelah bus API + scrapes the library's availability grid server-side
IntelligenceAgnes AI — natural-language dates & reminder schedules

🧩 Modular by design

One features/ file per tool (gym, deadlines, last_train, facilities). Add a feature = add a module + a menu button.

🛡️ Resilient

Generous timeouts, graceful Markdown fallback, reminders survive restarts, live fetches fail soft to a link.
